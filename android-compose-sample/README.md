# Exemplo de Navbar Android (Jetpack Compose)

Arquivo principal:
- MainActivity.kt

## O que este exemplo entrega
- Barra inferior preta com cantos arredondados.
- Tres itens: Inicio, Gaveta de Apps e Feed.
- Icones brancos nos itens inativos.
- Item ativo com circulo que sobe para fora da barra.
- Borda branca no circulo ativo.
- Icone ativo em laranja.
- Animacao suave na troca de selecao.

## Como usar no Android Studio
1. Crie um projeto Empty Activity com Jetpack Compose habilitado.
2. Substitua o conteudo de MainActivity.kt pelo arquivo deste exemplo.
3. Rode no emulador Android.

## Dependencias
Use Material 3 padrao de um projeto Compose atual.
